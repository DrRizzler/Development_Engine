# BaseEngine
Minimal C++ engine base using **SFML 3** for window/input and **OpenGL (via GLAD)** for rendering.

## Build (Linux)
Dependencies (Debian/Kali):
sudo apt update
sudo apt install -y build-essential cmake git libsfml-dev libgl1-mesa-dev
